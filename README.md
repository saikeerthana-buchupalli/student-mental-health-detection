**STUDENT MENTAL HEALTH DETECTION USING MACHINE LEARNING**
This project predicts the mental health risk level of students using machine learning.
*Features*
- Machine Learning model using Random Forest
- Flask backend API
- Web-based survey form
- Mental health risk prediction

*Technologies Used*
- Python
- Flask
- Scikit-learn
- HTML
- JavaScript
  
*How it Works*
1. Student fills the survey form.
2. Data is sent to Flask backend.
3. Backend loads trained ML model.
4. Model predicts mental health risk.
5. Result is displayed on the web page.

*Project Structure*
|
|--- app.py
|--- mental_health_model.pkl
|--- templates
   |---index.html

*Run the Project*
Install dependencies :
pip install flask scikit-learn pandas numpy
Run the app :
python app.py
Open in browser :
http://127.0.0.1:5000
